# chmods

> Have separate permissions set recursively for both files and directories

`chmod` on steroids for Unix-like systems.

### Usage

```
chmods 0.1.0
Have separate permissions set recursively for both files and directories

USAGE:
    chmods <entry-path>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <entry-path>
```
